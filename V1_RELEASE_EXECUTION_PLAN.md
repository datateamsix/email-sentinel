### Phase 1: Repository Setup (60 minutes)

#### Step 1.1: Create Homebrew Tap Repository

**What is a Homebrew Tap?**
A Homebrew tap is a GitHub repository containing Ruby formula files that tell Homebrew how to install your software on macOS.

**Instructions:**

1. **Create Repository on GitHub**
   - Go to: https://github.com/organizations/datateamsix/repositories/new
   - Repository name: `homebrew-tap`
   - Description: `Homebrew formulae for Email Sentinel`
   - **Visibility: PUBLIC** (required for Homebrew taps)
   - Initialize with README: Yes
   - Add .gitignore: None
   - License: MIT

2. **Clone and Setup**
   ```bash
   git clone https://github.com/datateamsix/homebrew-tap.git
   cd homebrew-tap
   ```

3. **Create README.md**
   ```bash
   cat > README.md << 'EOF'
   # Homebrew Tap for Email Sentinel

   Official Homebrew tap for [Email Sentinel](https://github.com/datateamsix/email-sentinel).

   ## Installation

   ```bash
   brew tap datateamsix/tap
   brew install email-sentinel
   ```

   ## Available Formulae

   - **email-sentinel** - Lightweight, smart real-time notification management system

   ## Updating

   ```bash
   brew update

#### Step 1.3: Generate GitHub Personal Access Tokens

**Why needed?**
GoReleaser needs permission to push formula/manifest files to your tap/bucket repositories.

**Instructions:**

1. **Generate Homebrew Token**
   - Go to: https://github.com/settings/tokens/new
   - Token name: `HOMEBREW_TAP_TOKEN`
   - Expiration: No expiration (or 1 year)
   - Scopes:
     - ‚úÖ `repo` (all sub-scopes)
     - ‚úÖ `workflow`
   - Click "Generate token"
   - **COPY THE TOKEN** (you won't see it again!)

2. **Generate Scoop Token**
   - Go to: https://github.com/settings/tokens/new
   - Token name: `SCOOP_BUCKET_TOKEN`
   - Expiration: No expiration (or 1 year)
   - Scopes:
     - ‚úÖ `repo` (all sub-scopes)
     - ‚úÖ `workflow`
   - Click "Generate token"
   - **COPY THE TOKEN**

**Pro Tip:** You can use the same token for both if you prefer (just copy it twice).

   - Value: [paste Docker Hub token]

## üÜò Troubleshooting

### Issue: GoReleaser fails with "repository not found"

**Solution:**
- Verify GitHub Secrets are correctly named (exact match)
- Check Personal Access Tokens have `repo` scope
- Ensure repositories are public

### Issue: Homebrew formula not created

**Solution:**
- Check `.goreleaser.yaml` has correct tap configuration:
  ```yaml
  brews:
    - repository:
        owner: datateamsix
        name: homebrew-tap
        token: "{{ .Env.HOMEBREW_TAP_TOKEN }}"
  ```
- Verify HOMEBREW_TAP_TOKEN is set in GitHub Secrets

### Issue: Docker push fails

**Solution:**
- Verify DOCKER_USERNAME and DOCKER_TOKEN in GitHub Secrets
- Check Docker Hub repository exists and is public
- Or switch to GitHub Container Registry (no secrets needed)

### Issue: GitHub Pages not loading

**Solution:**
- Wait 2-3 minutes after enabling
- Check Settings ‚Üí Pages shows "Your site is live at..."
- Verify `/docs` folder contains `index.html`
- Check Actions tab for Pages build workflow
