# Build
go build -o email-sentinel.exe .

# Run init
./email-sentinel.exe init
```

You should see:
1. It finds your `credentials.json`
2. Prints a Google OAuth URL
3. You open that URL, authorize, copy the code
4. Paste the code back
5. Token is saved to `%APPDATA%\email-sentinel\token.json`
```
